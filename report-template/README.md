# 报告模板概述

这里提供了针对不同开发板和示例的文档模板，旨在帮助贡献者记录和分享他们的测试结果。每个模板都包含了必要的结构和格式，以确保信息的清晰和一致性。

## 使用说明

1. **针对操作系统的测试报告模板**：位于 `report-template/[board-name]/report_os-name_board-name.md`，用于记录特定操作系统的测试环境、安装步骤、预期结果、实际结果及测试结论。
2. **针对开发板极速开发目录模板**：位于 `report-template/[board-name]/index_tutorial_board-name.md`，用于记录特定开发板的基本信息、环境配置、基础开发类示例、功能与库类示例、应用类示例的目录。
3. **针对开发板概述模板**：位于 `report-template/[board-name]/overview_board-name.md`，用于记录特定开发板的基本信息、硬件规格。
4. **针对开发板环境搭建模板**：位于 `report-template/[board-name]/boot_board-name.md`，用于记录特定开发板的环境搭建、RuyiSDK 环境初始化。
5.  **针对 Ruyi 示例模板**：位于 `report-template/[board-name]/example_example-name_board-name.md`，用于记录特定开发板的 ruyi 示例编译和运行。
6. **针对注意事项模板**：位于 `report-template/[board-name]/notes.md`，用于记录特定开发板的 ruyi 示例编译和运行。
7. **针对图片存放位置**：位于`report-template/[board-name]/[images]`，用于存放文档中的图片。

请根据具体的开发板和示例，填充和修改相应的模板内容。
